RandomAccessList<'T>

Namespace: FSharpx.Collections

RandomAccessList is an ordered linear structure implementing the List signature (head, tail, cons), as well as inspection (lookup) and update (returning a new immutable instance) of any element in the structure by index. Ordering is by insertion history. While PersistentVector<'T> is appending to the end this version prepends ekements to the list.
